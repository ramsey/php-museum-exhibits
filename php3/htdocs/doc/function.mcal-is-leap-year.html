mcal_is_leap_year

(PHP3 >= 3.0.13, PHP4 >= 4.0b4)

mcal_is_leap_year --  Returns if the given year is a leap year or not

Description

int mcal_is_leap_year (int year)

mcal_is_leap_year() returns 1 if the given year is a leap year, 0 if not.
