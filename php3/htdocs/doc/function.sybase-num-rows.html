sybase_num_rows

(PHP3 , PHP4 )

sybase_num_rows -- get number of rows in result

Description

int sybase_num_rows (string result)

sybase_num_rows() returns the number of rows in a result set.

See also: sybase_db_query(), sybase_query() and, sybase_fetch_row().
