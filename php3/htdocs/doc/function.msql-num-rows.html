msql_num_rows

(PHP3 , PHP4 )

msql_num_rows -- Get number of rows in result

Description

int msql_num_rows (int query_identifier)

Msql_num_rows() returns the number of rows in a result set.

See also: msql(), msql_query(), and msql_fetch_row().
