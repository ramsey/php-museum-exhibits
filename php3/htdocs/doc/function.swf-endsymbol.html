swf_endsymbol

(PHP4 >= 4.0RC2)

swf_endsymbol -- End the definition of a symbol

Description

void swf_endsymbol (void);

The swf_endsymbol() function ends the definition of a symbol that was started by the swf_startsymbol() function.
