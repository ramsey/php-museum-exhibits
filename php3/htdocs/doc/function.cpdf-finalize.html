(PHP3 >= 3.0.8, PHP4 >= 4.0b4)

cpdf_finalize -- Ends document

Description

void cpdf_finalize (int pdf document)

The cpdf_finalize() function ends the document. You still have to call cpdf_close()

See also cpdf_close().
