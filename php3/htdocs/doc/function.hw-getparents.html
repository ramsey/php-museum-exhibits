hw_GetParents

(PHP3 >= 3.0.3, PHP4 )

hw_GetParents -- object ids of parents

Description

array hw_getparents (int connection, int objectID)

Returns an indexed array of object ids. Each object id belongs to a parent of the object with ID objectID.
