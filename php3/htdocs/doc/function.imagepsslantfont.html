ImagePsSlantFont

(PHP3 >= 3.0.9, PHP4 >= 4.0RC1)

ImagePsSlantFont --  Slant a font

Description

bool imagepsslantfont (int font_index, double slant)

Slant a font given by the font_index parameter with a slant of the value of the slant parameter.
