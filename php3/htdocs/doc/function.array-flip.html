array_flip

(PHP4 >= 4.0b4)

array_flip -- Flip all the values of an array

Description

array array_flip (array trans)

Array_flip() returns an array in flip order.

Example 1. Array_flip() example

$trans = array_flip ($trans);
$original = strtr ($str, $trans);
