gettype

(PHP3 , PHP4 )

gettype -- Get the type of a variable

Description

string gettype (mixed var)

Returns the type of the PHP variable var.

Possibles values for the returned string are:

  • "boolean"

  • "integer"

  • "double"

  • "string"

  • "array"

  • "object"

  • "resource"

  • "user function"

  • "unknown type"

See also settype().
