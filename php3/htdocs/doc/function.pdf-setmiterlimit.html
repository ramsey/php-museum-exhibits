pdf_setmiterlimit

(PHP3 >= 3.0.6, PHP4 )

pdf_setmiterlimit -- Sets miter limit

Description

void pdf_setmiterlimit (int pdf document, double value)

The pdf_setmiterlimit() function sets the miter limit to a value greater of equal than 1.
