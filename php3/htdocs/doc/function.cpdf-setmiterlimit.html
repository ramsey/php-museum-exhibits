cpdf_setmiterlimit

(PHP3 >= 3.0.8, PHP4 >= 4.0b4)

cpdf_setmiterlimit -- Sets miter limit

Description

void cpdf_setmiterlimit (int pdf document, double value)

The cpdf_setmiterlimit() function set the miter limit to a value greater or equal than 1.
