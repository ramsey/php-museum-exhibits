ImageSetPixel

(PHP3 , PHP4 )

ImageSetPixel -- Set a single pixel

Description

int imagesetpixel (int im, int x, int y, int col)

ImageSetPixel() draws a pixel at x, y (top left is 0, 0) in image im of color col.

See also ImageCreate() and ImageColorAllocate().
