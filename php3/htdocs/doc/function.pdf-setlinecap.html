pdf_setlinecap

(PHP3 >= 3.0.6, PHP4 )

pdf_setlinecap -- Sets linecap parameter

Description

void pdf_setlinecap (int pdf document, int value)

The pdf_setlinecap() function sets the linecap parameter between a value of 0 and 2.
