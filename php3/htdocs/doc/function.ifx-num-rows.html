ifx_num_rows

(PHP3 >= 3.0.3, PHP4 )

ifx_num_rows -- Count the rows already fetched a query

Description

int ifx_num_rows (int result_id)

Gives the number of rows fetched so far for a query with result_id after a ifx_query() or ifx_do() query.
