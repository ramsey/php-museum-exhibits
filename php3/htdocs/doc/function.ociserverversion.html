OCIServerVersion

(PHP3 >= 3.0.4, PHP4 )

OCIServerVersion -- Return a string containing server version information.

Description

string OCIServerVersion (int conn)

Example 1. OCIServerVersion

<?php
   $conn = OCILogon("scott","tiger");
   print "Server Version: " . OCIServerVersion($conn);
   OCILogOff($conn);
?>
