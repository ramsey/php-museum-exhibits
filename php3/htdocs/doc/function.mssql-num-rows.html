mssql_num_rows

(PHP3 , PHP4 )

mssql_num_rows -- Get number of rows in result

Description

int mssql_num_rows (string result)

Mssql_num_rows() returns the number of rows in a result set.

See also: mssql_db_query(), mssql_query(), and mssql_fetch_row().
