dbmdelete

(PHP3 , PHP4 )

dbmdelete --  Deletes the value for a key from a DBM database

Description

bool dbmdelete (int dbm_identifier, string key)

Deletes the value for key in the database.

Returns false if the key didn't exist in the database.
