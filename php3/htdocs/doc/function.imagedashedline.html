ImageDashedLine

(PHP3 , PHP4 )

ImageDashedLine -- Draw a dashed line

Description

int imagedashedline (int im, int x1, int y1, int x2, int y2, int col)

ImageDashedLine() draws a dashed line from x1, y1 to x2, y2 (top left is 0, 0) in image im of color col.

See also ImageLine().
