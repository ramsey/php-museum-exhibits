mcal_delete_event

(PHP3 >= 3.0.13, PHP4 >= 4.0b4)

mcal_delete_event -- Delete an event from an MCAL calendar

Description

int mcal_delete_event (int mcal_stream [, int event_id])

mcal_delete_event() deletes the calendar event specified by the event_id.

Returns true.
