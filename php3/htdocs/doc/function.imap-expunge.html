imap_expunge

(PHP3 , PHP4 )

imap_expunge -- Delete all messages marked for deletion

Description

int imap_expunge (int imap_stream)

imap_expunge() deletes all the messages marked for deletion by imap_delete(), imap_mail_move(), or imap_setflag_full().

Returns true.
